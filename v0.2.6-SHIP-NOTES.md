## What shipped

### 1. Catatan Race — F1 podium share cards
Static PNG share cards for every completed Grand Prix in 3 variants:
- **og** 1200×630 — Open Graph / Twitter Card
- **story** 1080×1920 — Instagram Stories / WhatsApp Status
- **square** 1080×1080 — IG feed / X

Design mirrors the NBA recap card: dark gradient, `gibol.co` wordmark, team-color podium chips, Bahasa hook quote (4 rotating casual lines, selected by `round % 4`), F1-red accent footer. Fonts (Bebas Neue + Space Grotesk) base64-embedded in the SVG via `scripts/fonts/` for reproducible generator runs — no network at build time, no `@vercel/og` runtime.

Shipped for rounds 1-3 (9 PNGs at `public/og/f1/2026-R0{1,2,3}-{australian,chinese,japanese}-gp-{og,story,square}.png`). Rounds 4-5 (Bahrain, Saudi Arabia) will land the next time `--all-past` runs against a populated Jolpica.

`F1Race.jsx` auto-wires `og:image` when the race date has passed so WhatsApp / X / IG link unfurls show the podium card.

CLI:
```bash
node scripts/generate-f1-recap.mjs --all-past          # every completed GP
node scripts/generate-f1-recap.mjs --round 3           # single round
node scripts/generate-f1-recap.mjs --round 3 --variant story   # single variant
node scripts/generate-f1-recap.mjs --all-past --dry-run        # preview
```

### 2. Bilingual F1 news feed
New endpoint `GET /api/f1-news?lang=id|en&limit=15`:
- **ID sources:** detikSport, Bola.com, CNN Indonesia, Kompas
- **EN sources:** Autosport, Motorsport.com, BBC Sport, Formula1.com
- Native regex RSS 2.0 + Atom parser, no xml2js dependency
- 8s AbortController timeout across all fetches (one slow feed can't block the response)
- URL dedup, newest-first, cap to `limit`
- Cache: `s-maxage=900, stale-while-revalidate=1800`
- Returns 503 only if **all** sources failed AND items=0; partial success still returns 200

**Per `feedback_news_bilingual`: no machine translation.** The ID dashboard shows real Bahasa articles, the EN dashboard shows real English articles. Language toggle flips the source set, not the translator.

`useF1News(lang)` hook polls every 15 min. `F1News` component drops into the F1 dashboard as a third column alongside Driver + Constructor standings — terminal-style list with source chips (color per outlet) and relative timestamps ("2 jam lalu" / "2h ago").
