**Date:** 2026-04-20
**Branch:** main
**Sport flip:** `tennis` newly added → `'live'` (fourth sport LIVE alongside NBA + F1 + EPL)
**Ships ahead of:** Roland Garros 2026 (May 24 – Jun 7). 34-day pre-tournament SEO window live.

---

## What shipped

### 1. Hub `/tennis`

Bahasa-first ATP + WTA hub. Sections top-to-bottom:

- **Hero** — Inter Tight 36/700/-0.025em title, tennis gold accent `#D4A13A` tint (~8%), LIVE chip top-right.
- **CountdownToSlam** — next Grand Slam block with venue, date range, LIVE pulse when in-progress, compact days-to-go counter. Returns null post-US Open.
- **LiveTicker** — merged ATP + WTA scoreboard, priority-sorted (live > pre > post), limit 8.
- **TierSection** — one section per tier group: Grand Slam · Combined 1000 · Masters 1000 · WTA 1000 · Finals. TournamentCard previews link to per-tournament pages.
- **RankingsSnapshot** — top 10 ATP + top 10 WTA side-by-side with points.
- **IndonesianSpotlight** — Aldila Sutjiadi, Priska Madelyn Nugroho, Christopher Rungkat player cards.
- **TennisNewsList** — bilingual feed (Bahasa + English) from `/api/tennis-news?lang=id|en`.
- **Disclaimer + footer** — ESPN data-source credit + "not affiliated with ATP/WTA" note.

### 2. 18 per-tournament SEO pages `/tennis/{slug}-2026`

Each tournament indexable with its own `SportsEvent` JSON-LD. `useTennisTournament` hook parses slug → `{tournament, year, phase}`, then branches on phase:

- **UpcomingBody** — CountdownToSlam + TournamentFactSheet + `humanRelativeDays` countdown copy.
- **LiveBody** — TournamentFactSheet + ATP/WTA scoreboard filtered to matches whose `nameTokens.length > 3` match the tournament.
- **CompletedBody** — TournamentFactSheet + archive note.

TournamentFactSheet exposes venue / city / dates / tours / organizer / web as Fact cards. RelatedTournaments chip strip at the bottom closes the internal link graph. Invalid slug → `Navigate` to `/tennis`.

**Slugs** (18): `australian-open-2026, roland-garros-2026, wimbledon-2026, us-open-2026, indian-wells-2026, miami-open-2026, madrid-open-2026, italian-open-2026, canadian-open-2026, cincinnati-open-2026, monte-carlo-2026, shanghai-masters-2026, paris-masters-2026, dubai-championships-2026, china-open-2026, wuhan-open-2026, wta-finals-2026, atp-finals-2026`.

### 3. Rankings `/tennis/rankings/:tour`

`:tour` is `atp` or `wta` — invalid values redirect to `/tennis/rankings/atp`.

- **TourSwitcher** — ATP / WTA pill toggle.
- **TopPodium** — top-10 grid cards (rank, country code, name, points).
- **IndonesianInline** — one-line highlight when any `IDN`/`INA`/`ID` country code is in top 500.
- **RankingsTable** — 5-col grid, limit 100, weekly up/down trend arrows.

### 4. Data layer

Four new hooks routing through the existing edge-cached `/api/proxy/espn/*` provider (no new proxy providers needed):

| Hook | Endpoint | Cache | Polling |
|---|---|---|---|
| `useTennisScoreboard` | `espn/tennis/atp/scoreboard` + `espn/tennis/wta/scoreboard` | s-maxage=60 | 60s |
| `useTennisRankings` | `espn/tennis/{tour}/rankings` | s-maxage=300 | one-shot (weekly refresh) |
| `useTennisNews` | `/api/tennis-news?lang=id|en` | s-maxage=900 | 5min |
| `useTennisTournament` | client-only slug parser against `tournaments.js` registry | n/a | n/a |

`api/tennis-news.js` aggregates Bahasa (detikSport + Antara + CNN Indonesia) and English (ESPN + BBC Sport + Tennis.com) RSS feeds. Same regex-based parser and cache strategy as `/api/f1-news`. No machine translation.

### 5. 8 new components under `src/components/tennis/`

`SurfaceChip` · `TierChip` · `ScoreLine` (per-set with tiebreak `<sup>` + server dot) · `PlayerChip` (seed · name · CC, Indonesian tint) · `LiveMatchCard` · `TournamentCard` · `RankingsTable` (5-col grid, trend arrows, Indonesian-CC row tint) · `CountdownToSlam` (full + compact variants).

### 6. SEO surface

- `sitemap.xml` — **21 new tennis URLs**: 1 hub @ priority 0.9, 2 rankings @ 0.8, 18 per-tournament weighted by prestige (Roland Garros 0.9; Australian Open / Wimbledon / US Open 0.8; ATP Finals + WTA Finals 0.7; Masters 1000 / WTA 1000 / early-500s 0.6).
- `adapter.prerenderRoutes()` — returns 21 route metadata objects. Generic `scripts/prerender.mjs` auto-iterates — no prerender script edits needed.
- `llms.txt` — Tennis moved to "Live dashboards" with its own Key facts block (surfaces, slugs, ranking URLs, Indonesian players, data source). Also moved EPL from "Coming soon" → "Live dashboards" (was stale — EPL went live in v0.4.0).
- `Home.jsx` — 6th dashboard card at `#D4A13A` accent, LIVE tag, blurb *"4 Grand Slams · Masters 1000 · WTA 1000 · Year-End Finals. Live sets, ATP + WTA rankings, Indonesian player spotlight."* SEO keywords extended with `tenis / atp / wta / grand slam / roland garros / peringkat`.

### 7. Tennis color tokens

9 new tokens on `COLORS` — `tennisSlamGold` (`#D4A13A`), `tennisClayRust`, `tennisGrassGreen`, `tennisHardBlue`, `tennisMastersSilv`, `tennisLive`, `tennisAccent`, surface chip tints. Zero hardcoded hex in components.
