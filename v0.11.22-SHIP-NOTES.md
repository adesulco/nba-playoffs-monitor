## What shipped

### GIB-005 тАФ тМШK listener strengthened
`src/components/v2/TopBar.jsx`. Three changes:
- Listener now binds on `document` in **capture phase** (was `window` bubble). If anything upstream `stopPropagation`s a `keydown`, our handler still fires.
- Added `e.repeat` guard so held-down keys don't queue multiple opens.
- Skipped when `e.target` is `contentEditable` or a `<textarea>` so the shortcut doesn't hijack a user typing in their own input.

The audit's "nothing happens" path was likely a stopPropagation upstream (now bypassed) or a Chrome-extension shortcut interception (out of our scope). Code path is now defensively maximal.

### GIB-006 тАФ search catalogue + matcher fix
`src/components/v2/SearchPalette.jsx`.

**Matcher.** `filterRows` now also matches against `short` (the abbr / 3-letter code). Before, the power-user query "LAL" returned zero rows because the field wasn't searched. Now it surfaces "Los Angeles Lakers." Same for "VER" тЖТ Verstappen, "ARS" тЖТ Arsenal, etc.

**Catalogue.** Was NBA-only (16 entries). Now imports from canonical constants:

| Source | Rows |
|---|---|
| NBA teams (in-file table) | 16 |
| EPL clubs (`CLUBS` from `epl/clubs.js`) | 20 |
| F1 constructors (`TEAMS_2026` from `f1/constants.js`) | 11 |
| F1 drivers (`DRIVERS_2026` from `f1/constants.js`) | 22 |
| Tennis stars (`TENNIS_STARS_BY_SLUG` from `tennis/constants.js`) | ~15 |
| **Total** | **~84** |

Each row links to the appropriate destination тАФ per-team / per-driver SEO pages for NBA / EPL / F1, and picker-pre-filtered hub URLs for Tennis (e.g. `/tennis?player=carlos-alcaraz`, leveraging v0.11.8's `useQueryParamSync`).

### GIB-022 тАФ search empty-state hint
`src/components/v2/SearchPalette.jsx`. Added a one-line banner above the Trending block when the query box is empty:

> **Try:** Lakers ┬╖ OKC ┬╖ Arsenal ┬╖ Verstappen ┬╖ Alcaraz ┬╖ Sinner

Tells the user what's searchable on first open. Audit's Diandra persona never reached the LAL test because the modal looked sparse тАФ this fixes the affordance.
