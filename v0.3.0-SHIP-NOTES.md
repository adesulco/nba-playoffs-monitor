### 2. Typography — Inter Tight + JetBrains Mono
- `index.html` font URL swapped from `Inter / JetBrains Mono / Space Grotesk / Bebas Neue` → **`Inter+Tight:wght@400;500;600;700;800;900` + `JetBrains+Mono:wght@400;500;700`**.
- Body `font-family` now `var(--font-sans)` (Inter Tight). Numerics use `.mono` / `.tabular` utility classes.
- Mass-replaced inline `'"Space Grotesk", sans-serif'` / `'"Bebas Neue", sans-serif'` → `'var(--font-sans)'` across 24 files (.jsx / .js).
- New utility classes in `index.css`:
  - `.brand-wordmark` — Inter Tight 900, `-0.055em` tracking
  - `.brand-wordmark-dot` — forces the final `.` to `--amber`
  - `.big-score` — Inter Tight 900, `-0.06em`
  - `.card-title` — 11px / 0.14em uppercase in `--ink-3`
  - `.mono` / `.tabular` — JetBrains Mono with `font-variant-numeric: tabular-nums`

### 3. Logo — target-mark glyph + typographic wordmark
New `src/components/Logo.jsx` replaces the PNG-based lockup:
- `<Glyph>` — inline SVG (concentric circles + tick marks) using `currentColor` so the mark inherits theme.
- `<Logo>` — full lockup = Glyph + `gibol` Inter Tight 900 wordmark + amber-dot final period. `size="sm"|"md"|"lg"`, `glyphOnly` prop for ≤32px surfaces.
- Clearspace = glyph height (baked into inter-element gap).
- Minimums: 20px full lockup, 16px glyph-only.

`TopBar.jsx` masthead swapped from `<img src="/favicon-64.png" />` → `<Glyph size={28} />` + `gibol.` wordmark. No other component changes.

### 4. Favicons + PWA icons regenerated
Rasterized from the new glyph SVG on a navy tile (stroke = `--ink`, center dot = `--amber`):

## QA checklist (post-deploy)

```bash
# 1. Fonts loaded fresh from Google
curl -sI "https://fonts.googleapis.com/css2?family=Inter+Tight:wght@400;500;600;700;800;900&family=JetBrains+Mono:wght@400;500;700&display=swap" | head -3

# 2. theme-color reflects in HTML source
curl -s https://www.gibol.co/ | grep 'theme-color'

# 3. Favicons resolvable
for f in favicon-16 favicon-32 favicon-64 apple-touch-icon gibol-logo-512; do
  curl -sI "https://www.gibol.co/$f.png" | head -1
done

# 4. Manifest icons
for f in icon-192 icon-512 icon-maskable-192 icon-maskable-512; do
  curl -sI "https://www.gibol.co/icons/$f.png" | head -1

curl -sI https://www.gibol.co/ | head -1
curl -sI https://www.gibol.co/nba-playoff-2026 | head -1
curl -sI https://www.gibol.co/formula-1-2026 | head -1
```
