## What shipped

- **GIB-007** Nav labels translate now. `NAV_ITEMS` in `V2TopBar.jsx` carries a `labelKey` per item; `t(labelKey)` resolves via `i18n.js`. Bahasa users see `Beranda · NBA · Bola · F1 · Tenis · Piala Dunia`. NBA + F1 stay as proper-noun acronyms.
- **GIB-008** `World CupSOON` → `World Cup SOON`. Inserted explicit `{' '}` between the label and the SOON badge span so `textContent` reads with a space. Visual gap was already drawn by the parent flex's `gap`; this only changes what AT and analytics see.
- **GIB-009** `LIVLiverpool` + the same pattern fixed across all four sport pickers — `TeamPicker.jsx` (NBA), `ConstructorPicker.jsx` (F1), `EPLClubPicker.jsx` (EPL), `TennisPlayerPicker.jsx` (Tennis). One JSX whitespace per file.
- **GIB-010** `SEASON 2026` → `t('season') 2026`. New keys `season: 'SEASON'` (EN) / `season: 'MUSIM'` (ID) in `i18n.js`. Replaced hardcodes in `F1.jsx` and `Tennis.jsx` hero strips.
- **GIB-011** Home, HomeV1, and NBADashboard now pass `title={lang === 'id' ? '…' : '…'}` and `description={lang === 'id' ? '…' : '…'}` into `<SEO>`. Helmet's `<title>` sync runs on every prop change, so toggling BI/EN mid-session updates `document.title` cleanly. FIFA / Liga 1 / IBL deferred — they pipe through ComingSoon's static SEO and get picked up in the Day 5 OG-image work.
