# v0.11.17 тАФ panel-title voice unified (Bloomberg mono)

**Date:** 2026-04-24
**Risk:** LOW. Visual-only change: adds `className="panel-title-mono"` to 12 h2s across three sport hubs. No data path, no layout. CSS class shipped since v0.11.12.

---

## Why

Audit finding from 2026-04-23 live sweep: NBA dashboard panel titles read as 10 px UPPERCASE JetBrains Mono since v0.1 ("TITLE ODDS", "ROUND 1 BRACKET", "WATCHLIST"); EPL / F1 / Tennis shipped with 15 px Inter Tight Title Case ("Title odds", "Championship odds", "Top Scorers"). The mix made the four sport hubs feel like four different products stapled together.

v0.11.12 shipped `.panel-title-mono` as a reserved utility class but didn't wire it pending a design decision. That decision locked in: Bloomberg-terminal voice wins тАФ it's the original identity, matches the `--font-mono` token already in the brand handoff, and is more distinctive than the Inter-Tight-everywhere direction.

## What shipped

`className="panel-title-mono"` added to every panel h2 on the three non-NBA sport hubs:

- **EPL** (`src/pages/EPL.jsx`, 7 h2s): Table ┬╖ Title odds ┬╖ This Week's Fixtures ┬╖ Latest Results ┬╖ Top Scorers ┬╖ Club Pages ┬╖ News
- **F1** (`src/pages/F1.jsx`, 3 h2s): Drivers Standings ┬╖ Constructors Standings ┬╖ Championship odds
- **F1News** (`src/components/F1News.jsx`, 1 h2): F1 News
- **Tennis** (`src/pages/Tennis.jsx`, 2 h2s): Title odds ┬╖ {Slam} ┬╖ Active & upcoming tournaments (composites with `tick-live` when any tournament is live)

The `.panel-title-mono` class uses `!important` on font-family / size / weight / letter-spacing / text-transform so the inline React `style={{}}` props each consumer still sets (for per-page colour and margin) don't fight the voice. Same technique NBA's custom panelTitle inline style has always used.

Computed style for every panel h2 on EPL / F1 / Tennis is now:
`JetBrains Mono ┬╖ 700 ┬╖ 12 px ┬╖ letter-spacing 1.5 ┬╖ uppercase ┬╖ color --ink`

NBA's existing 10 px inline labels stay unchanged тАФ already Bloomberg-terminal.
